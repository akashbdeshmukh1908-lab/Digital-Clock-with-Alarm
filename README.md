# Digital Clock with Alarm

## Intern Details



## Project Scope

The Digital Clock with Alarm project is designed to display the current system time in real-time and provide alarm functionality for users. The application offers an easy-to-use graphical interface where users can set alarm times and receive notifications when the specified time is reached. This project demonstrates the use of Java Swing for GUI development, multithreading for continuous time updates, and event handling for alarm management.

## Features

* Real-time Digital Clock
* 12-Hour and 24-Hour Time Format
* Alarm Setting Functionality
* Alarm Notification Message
* User-Friendly GUI
* Start and Stop Alarm
* Accurate Time Synchronization with System Clock

## Technologies Used

* Java
* Swing
* AWT
* Multithreading
* Object-Oriented Programming

## How to Run

1. Install Java JDK.
2. Compile the program:

   ```
   javac DigitalClockWithAlarm.java
   ```
3. Run the program:

   ```
   java DigitalClockWithAlarm
   ```
4. Set the desired alarm time and click the Set Alarm button.

## Output

The application displays the current time continuously and alerts the user when the alarm time matches the system time.

## Conclusion

This project demonstrates the implementation of real-time clock functionality and alarm scheduling using Java. It helps in understanding GUI design, multithreading, and event-driven programming concepts.
